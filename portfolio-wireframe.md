## Key Features:
1. **Smart Filtering**: Projects/skills/certs change based on selected path
2. **Unified View Option**: "View All" shows complete capabilities
3. **Freelance Mode**: Shows rates, availability, consultation booking
4. **Corporate Mode**: Emphasizes team fit, enterprise experience
5. **Visual Timeline**: Shows progression across all paths
6. **Beyond Code**: Humanizes you for better connections

## Technical Implementation:
- React/Vue for dynamic content switching
- URL parameters for direct linking (site.com/?path=aiml)
- Analytics to track which paths get most views
- Fast switching with no page reloads
- Progressive enhancement (works without JS)
